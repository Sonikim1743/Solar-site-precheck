## Version 1.1 — 2026-06-30

Solar Site Precheckを太陽光候補地チェックだけでなく、周辺業務の調査補助にも使えるようにするため、法務局・相続関連PDFの読取補助機能を追加しました。

### 1. 登記・相続資料PDFチェック機能

- ページ下部に `相続資料PDFチェック（実験機能）` を追加。
- 法務局・相続関連PDFをブラウザ内で読み取り、土地の単独相続候補を抽出するプロトタイプを実装。
- `第○号 / ○月○日受付 / 単独 / 所有権移転・相続 / 土地 / 所在 / 外○件` 形式の要約行を抽出。
- PDFはサーバー送信・自動保存せず、ローカルブラウザ内で処理。

### 2. 実務リスト・Excel出力

- 画面表示は判定や点数を出さず、`受付番号 / 受付日 / 土地 / 住所 / 外記載` の実務リストに整理。
- 候補件数は `単独 / 所有権移転・相続（土地）` の件数として直感的に表示。
- 受付番号、受付日、土地、住所、外記載をExcel用CSVとして出力。
- 相続資料PDFチェックの抽出リストに、受付順・外記載が多い順・住所順の並び替えと、行単位コピー機能を追加。

### 3. PDF読取精度・全件確認

- PDF.jsで分割された文字列を座標から行単位に再構成し、`土地住所外N` のように空白がない行でも外記載を読めるよう改善。
- 抽出結果を80件で打ち切る制限を廃止し、`第○号` から次の `第○号` までの受付ブロックを最後まで出力するよう改善。
- PDF全体の受付番号について、最初の番号・最後の番号・読取件数・範囲件数・抜け番候補を表示し、全件読取できているか確認できるよう改善。
- 抜け番候補について、黒塗り・取下・受付文字なし・表示形式違い等で説明できる候補数を照合し、未説明候補が残るか確認できるよう改善。

### 4. 検証・制限

- 三次 R8.3 相続PDFで、受付番号範囲・読取件数・抜け番分類の動作を確認。
- Version 1.1更新時点で、テスト24件が全て通過。
- ビルド成功を確認。
- 現時点ではテキスト抽出可能なPDFを対象とし、スキャンPDFのOCRは今後の検討事項。

## Version 1.01 — 2026-06-29

Solar Site Precheckをチーム内検証用ツールとして使いやすくするため、Version 1.00から以下を改善しました。

### 1. 候補地点選択の簡略化

- 住所検索と緯度経度検索を1つの入力欄に統合。
- 入力内容が座標形式の場合は直接その地点へ移動し、住所・地名の場合は住所検索を行う方式に変更。
- 地番ファイル読込・筆界確認などの任意機能は「追加機能」にまとめ、通常時の画面を簡潔化。

### 2. 候補地切替時の入力残りを防止

- 新しい地点を選択した場合、候補地名、メモ、現地確認メモ、選択地番、隣接メッシュ比較結果などを初期化。
- 過去の案件情報が残って誤入力につながるリスクを低減。
- 候補地名欄には、選択地点から取得した周辺住所を補助情報として表示。

### 3. NEDO 3次メッシュ確認の見やすさ改善

- 3次メッシュ境界確認を大きな常時表示カードから、状態ランプ型の表示に変更。
- 境界から十分離れている場合は青色の「境界距離 OK」として表示。
- 境界100m以内の場合は黄色の「境界近接」とし、クリック時のみ隣接3次メッシュ比較を展開。
- 通常時の画面占有を減らし、必要な時だけ比較情報を確認できるよう改善。

### 4. 地平線分析表示の改善

- 通常の簡易分析を7方位から8方位へ変更し、北西315°を追加。
- 詳細36方位分析時に、方位名が空欄の角度で空の括弧が表示される問題を修正。
- 最大地平線角について、方位角・方位名・影響しやすい時間帯を表示。
  - 例：`分析済み：最大 23.0°（110° 東南東・午前側）`
- 想定樹高を変更した際に、分析結果が一時的に消える挙動を改善。

### 5. 積雪・発電量係数表示の改善

- 積雪10cm以上出現率の強調表示を整理。
- `0.01以上` は着色、`0.50以上` は **❄ 積雪注意** として強調。
- 発電量係数の説明を、公式部分と注意表示部分に分けて読みやすく改善。

### 6. モバイル表示・レイアウト改善

- 詳細地平線分析時、カードが下のセクションに重なって表示される問題を修正。
- Solar Pro入力画面プレビューやボタンが浮いて見える部分を、自然に下へ流れるレイアウトに調整。
- 候補地チェックレポートや補助情報の折りたたみ表示を整理。

### 7. Solar Pro転記・教育用途の補強

- Solar Pro入力画面スクリーンショットの表示位置を調整。
- 設置場所、地平線、積雪補正の入力画面説明を追加。
- Solar Pro入力マニュアルを折りたたみ式セクションとして整理。
- 情報・ダウンロード・リンク集を追加し、参考GIS・ハザード・農地・文化財・登記所備付地図データへの導線を整理。

### 8. 安定性・検証

- 主要な純粋関数についてテストを追加・維持。
- Version 1.01更新時点で、テスト17件が全て通過。
- ビルド成功を確認。

### 補足

本ツールはSolar Proを置き換えるものではなく、Solar Pro入力前の候補地確認・入力準備・社内共有を支援するための補助ツールです。Version 1.01では、実務検証で見つかった表示の分かりにくさ、入力残り、モバイル表示、3次メッシュ確認の見やすさを中心に改善しました。
