Solar Site Precheck 配布ガイド

この軽量更新ZIPには、Portable更新用ファイルに加えて Cloudflare Pages 用の functions/api と設定メモを同梱しています。

Portable版を更新する場合

  1. 既存のSolar Site Precheckフォルダを開きます。
  2. UPDATE_APP_FROM_RELEASE.cmd を実行します。
  3. 起動後、画面下部の「実行環境・API診断」でVersion / Build / API状態を確認します。

Cloudflare Pagesへ反映する場合

ZIPを展開したフォルダで、Wranglerを使って以下を実行します。

npx wrangler pages deploy dist --project-name=solar-site-precheck --functions=functions

Cloudflare Pagesのドラッグ&ドロップ配布では functions が反映されません。NEDO Web取得を使う場合はWrangler配布を使ってください。

機能差の目安

  • Cloudflare Pages: HTTPSで安定公開。NEDO APIはFunctionsで利用可能。相続PDFサーバー解析は別APIが無い限り利用不可。
  • Portable / Local: NEDO APIと相続PDFサーバー解析をローカルNodeで利用可能。PDF目安サイズは80MB。
