# 法務省・地番レイヤーの使い方

## 1. 元データを取得

法務省がG空間情報センターを通じて公開する「登記所備付地図データ」を、市区町村単位で取得します。

- 法務省案内：https://www.moj.go.jp/MINJI/minji05_00494.html
- G空間情報センター：https://front.geospatial.jp/

ダウンロード時に表示される利用規約を確認してください。

## 2. ダウンロードしたZIPまたはXMLを読み込む

外部の変換ソフトは不要です。G空間情報センターから取得したZIPは、展開せずそのままアプリへ読み込めます。展開済みの地図XML、従来のGeoJSONにも対応しています。

アプリ内の変換処理は、デジタル庁が公開する変換仕様を参考にしています。

- デジタル庁の変換ツール：https://github.com/digital-go-jp/mojxml2geojson

## 3. アプリへ読み込む

1. 画面の「XML・ZIP・GeoJSONを読み込む」を押します。
2. 50MBを超えるGeoJSONでは、先に住所検索または地図クリックで候補地点を選びます。
3. ダウンロードした `.zip`、展開済みの `.xml`、または `.geojson` / `.json` を選択します。
4. 大容量GeoJSONは、候補地点から3km以内の筆だけを自動抽出します。
5. 航空写真に黄色の筆界が表示されます。
6. 地番検索欄へ地番を入力するか、筆ポリゴンをクリックします。
7. 選択した地番の代表位置が候補地点になり、レポートとCSVへ反映されます。

## 制限事項

- 公共座標系の「14条1項地図」は航空写真へ重ねられます。
- 任意座標系の「地図に準ずる図面（公図、14条4項地図）」は実世界の位置座標がなく、アプリでは除外されます。
- ZIPに複数のXMLが含まれる場合、公共座標系のファイルだけをまとめて表示し、変換できなかったファイル数を画面に表示します。
- 公開データは現地境界の法的確定や測量を代替しません。正式判断では地図証明書、登記事項、現地測量を確認してください。
- 1ファイル300MBまで対応します。50MBを超えるGeoJSONは、画面の負荷を抑えるため選択地点の周辺だけを表示します。
